⚡ LangChain + Groq (Node.js/TS)

  Una implementación de alto rendimiento para practicar la orquestación de LLMs utilizando el ecosistema de LangChain.js y la inferencia ultra rápida de Groq Cloud.

🚀 ¿Por qué Groq?

Esta práctica se centra en la baja latencia. Utilizamos el SDK de LangChain para conectar con la API de Groq, permitiendo flujos de IA casi instantáneos con modelos como Llama 3 o Mixtral.
🎯 Objetivos de la Práctica


  ChatGroq Integration: Configuración del cliente oficial de LangChain para Groq.

  LCEL (LangChain Expression Language): Composición modular mediante .pipe().

  Prompt Engineering: Estructuración de ChatPromptTemplate para modelos Open Source.

  type Safety: Implementación estricta con TypeScript.

🛠️ Configuración Rápida

  Instalar dependencias:
   

    npm install @langchain/groq @langchain/core dotenv

  Variables de Entorno (.env):
  Code snippet

    GROQ_API_KEY=gsk_tu_clave_aqui

  Ejecución:
 

     npm run dev
